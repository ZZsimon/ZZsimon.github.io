# 新麦可视化页面搭建系统后台



```js

//本地全局安装@tarojs/cli@3.0.28 特定版本,另外开启一个终端
npm install -g @tarojs/cli@3.0.28


// 本地开发
yarn dev:h5

// 打包静态文件
yarn build:h5

// 上传文件
yarn deploy

// 本地启动node服务器，访问打包后的静态文件
yarn dev-server
```
